## v1.1.0 - 2026-05-30

### 新增功能：HTTP/HTTPS 域名映射（vhost）

支持基于域名的 HTTP/HTTPS 内网穿透，多个代理可共享同一端口（80/443），通过域名区分路由。

#### 服务端变更

- **新增 HTTP vhost 代理**：`HTTPVhostProxy` 类，根据 HTTP 请求的 `Host` 头路由到对应客户端
- **新增 HTTPS vhost 代理**：`HTTPSVhostProxy` 类，服务端 TLS 终止后按 `Host` 头路由
- **新增 VhostHTTPServer**：共享 HTTP 端口（默认80），按域名分发请求
- **新增 VhostHTTPSServer**：共享 HTTPS 端口（默认443），支持用户指定 SSL 证书
- **FrpServer 新增参数**：
  - `vhost_http_port` — HTTP vhost 监听端口（0=不启用）
  - `vhost_https_port` — HTTPS vhost 监听端口（0=不启用）
  - `cert_file` — HTTPS 证书文件路径（PEM 格式）
  - `key_file` — HTTPS 私钥文件路径（PEM 格式）
  - `subdomain_host` — 子域名根（如 `frp.example.com`）
- **命令行新增参数**：`--vhost-http-port`、`--vhost-https-port`、`--cert-file`、`--key-file`、`--subdomain-host`
- **HTTP Basic Auth 支持**：vhost 代理可配置 `http_user` / `http_pwd` 进行访问认证
- **子域名支持**：客户端可配置 `subdomain` 前缀，服务端自动拼接 `subdomain.subdomain_host`

#### 客户端变更

- **支持 http/https 代理类型注册**：`_register_proxy` 新增 HTTP/HTTPS 处理分支
- **注册消息新增字段**：`custom_domains`、`subdomain`、`http_user`、`http_pwd`、`host_header_rewrite`
- **示例配置新增** HTTP/HTTPS 代理模板

#### GUI 变更

- **服务端 GUI**：
  - 新增 vhost 配置项：HTTP端口、HTTPS端口、证书文件、私钥文件、子域名根
  - 新增证书/私钥文件浏览按钮
  - 保存/加载配置支持 vhost 字段
  - 代理列表 HTTP/HTTPS 代理显示域名而非端口号
- **客户端 GUI**：
  - 代理类型下拉框新增 `http`、`https` 选项
  - 新增字段：自定义域名、子域名前缀、HTTP用户名、HTTP密码
  - 代理列表 HTTP/HTTPS 显示域名信息
- **服务管理器**：
  - 新增 vhost 配置界面
  - 服务模式启动传递 vhost 参数

#### 配置文件变更

- `frps_service_config.json` 新增字段：`vhost_http_port`、`vhost_https_port`、`cert_file`、`key_file`、`subdomain_host`
- `frpc_config.json` 新增 HTTP/HTTPS 代理示例

#### 使用示例

```json
// 服务端配置
{
  "bind_addr": "0.0.0.0",
  "bind_port": 7000,
  "token": "your-token",
  "vhost_http_port": 80,
  "vhost_https_port": 443,
  "cert_file": "/path/to/cert.pem",
  "key_file": "/path/to/key.pem",
  "subdomain_host": "frp.example.com"
}

// 客户端代理配置
{
  "name": "my_web",
  "type": "http",
  "local_ip": "127.0.0.1",
  "local_port": 8080,
  "custom_domains": ["www.example.com", "example.com"]
}
```

## v1.0.0 - 2026-05-28

### 初始发布

- 基于 [fatedier/frp](https://github.com/fatedier/frp) 通信协议二次开发
- 纯 Python 实现 frp 协议，无额外依赖
- TCP 端口映射代理
- UDP 数据包转发代理
- XTCP NAT 穿透（P2P 直连）
- Token 认证
- 心跳检测与自动重连
- Windows 系统服务管理（注册/卸载/启动/停止）
- 服务端图形化管理界面（tkinter）
- 客户端图形化管理界面（tkinter）
- 工作连接池预分配
- 单文件 EXE 打包分发
